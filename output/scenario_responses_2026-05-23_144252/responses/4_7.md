Using `==` with strings can be tricky because it checks if both variables point to the same memory location, not if they have the same value. 

Here's an example:

```java
String s1 = "hello";
String s2 = "hello";

if (s1 == s2) { // This will return false
    System.out.println("They are equal");
} else {
    System.out.println("They are not equal");
}

if (s1.equals(s2)) { // This will return true
    System.out.println("They are equal using equals()");
}
```

In this case, `s1` and `s2` have the same value but not the same memory location. So, the first condition with `==` returns false, while the second condition with `.equals()` returns true.

Using `.equals()` ensures you're comparing the actual values of strings, not just their memory locations.

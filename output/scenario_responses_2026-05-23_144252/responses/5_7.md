That's correct! In Java, when you compare two string objects using `==`, you're actually checking if both variables are referencing the same instance in memory, not if their contents are equal.

Here's an example to illustrate this:

```java
String s1 = "Hello";
String s2 = "Hello";

System.out.println(s1 == s2); // false

String s3 = new String("Hello");
System.out.println(s1 == s3); // false
```

As you mentioned, the `equals()` method is what we want to use when comparing the actual contents of two strings:

```java
System.out.println(s1.equals(s2)); // true
System.out.println(s1.equals(s3)); // true
```

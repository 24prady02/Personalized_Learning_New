When a variable is "undefined," it means Java doesn't know what value it holds. This usually happens when you try to use a variable before declaring or initializing it.

Let's take an example:

```java
int x = 5;
System.out.println(x * y); // Error! y is undefined
```

```java

int y = 10; // Declare and initialize y
System.out.println(x * y); // Now it works!
```

## Ollama response
That's a common issue! In Java, variables declared inside an `if` block are only accessible within that block. If you want to use the variable outside of it, declare it before the `if` statement.

For example:

```java
int x;
if (someCondition) {
    x = 5; // Declare and assign here
} else {
    x = 10;
}
// Now you can use x outside the if block
System.out.println(x); // prints either 5 or 10
```

If you're trying to access a variable that's declared inside an `if` block from another method, consider declaring it at a higher scope, like at class level. That way it'll be accessible everywhere.

You're right that in Java, we typically use the `equals()` method to compare Strings. However, there's an important nuance here. The reason we use `equals()` is because of how Java handles String comparison behind the scenes.

Try comparing two String objects using a simple "==" operator:
```java
String str1 = new String("hello");
String str2 = new String("hello");

System.out.println(str1 == str2); // Output: false
```
As you can see, even though `str1` and `str2` have the same value, they are not considered equal because Java sees them as two separate objects. 

Using `equals()` method will give us the correct result:
```java
String str1 = new String("hello");
String str2 = new String("hello");

System.out.println(str1.equals(str2)); // Output: true
```

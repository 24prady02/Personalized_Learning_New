## Ollama response
You're catching on quickly! Yes, `==` in Java is indeed a reference comparison, not a value comparison like you might expect. To compare values, use the `.equals()` method instead, like this:

```java
String s1 = "hello";
String s2 = new String("hello"); // different objects, same contents

if (s1 == s2) {
    System.out.println("References are equal");
} else if (s1.equals(s2)) {
    System.out.println("Values are equal");
}
```

This code will print "Values are equal". So don't worry about not knowing this before - it's a common source of confusion, and now you do! Just remember to check the Java API docs for each class and method to see what their methods actually do.

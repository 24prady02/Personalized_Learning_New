You're on the right track! When comparing strings in Java, it's common to see both `==` and `.equals()` used. However, there's a subtle difference between them.

`==` checks if two objects are referencing the same memory location, whereas `.equals()` checks for equality of their actual values.

Consider this example:

```java
String str1 = "Hello";
String str2 = "Hello";

System.out.println(str1 == str2); // prints: false (because they're not in the same memory location)

System.out.println(str1.equals(str2)); // prints: true (because their values are equal)
```

In most cases, you'll want to use `.equals()` to compare strings because it's more intuitive and reliable. So go ahead and stick with `str1.equals(str2)`.

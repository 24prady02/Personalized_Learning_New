## Ollama response
The single equals sign is for assignment, whereas we want a comparison in this case. We use double equals signs for equality checks like `if (x == 5)`. This will evaluate to true only when the value of x is exactly 5.

```java
int x = 3;
if (x == 5) {
    System.out.println("x is equal to 5");
} else {
    System.out.println("x is not equal to 5");
}
```

In this example, "x is not equal to 5" will be printed because x is assigned the value 3.

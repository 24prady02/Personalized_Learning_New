Using `==` for string comparison is a common mistake, but it won't work as expected in Java. When you use `==`, it checks if both variables refer to the same memory location, not their actual values.

For example:
```java
String s1 = "hello";
String s2 = new String("hello");

System.out.println(s1 == s2);  // prints: false (even though they have the same value)
```
To compare strings correctly, use the `equals()` method:
```java
String s1 = "hello";
String s2 = "hello";

System.out.println(s1.equals(s2));  // prints: true
```
